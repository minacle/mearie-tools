# attach_images

이미지를 첨부하도록 유도하는 함수 도구입니다.

## 입력

**JSON**:

- `image_urls` (_array\[string\]_): 첨부할 이미지의 URL 목록입니다. 파일을 첨부하려면 `file://` 스킴을 사용합니다.

## 출력

**JSON**:

- `@mearie` (_object_): 메아리 지시 개체입니다.
  - `action` (_string_): 항상 `image.attach`입니다.
  - `data` (_array\[string\]_): 첨부할 이미지의 URL 목록입니다.

## 동작

첨부할 이미지의 URL 목록을 받아 메아리 지시 개체로 변환하여 반환합니다.

이 단계에서는 실제로 이미지를 첨부하지는 않지만, 메아리가 이미지를 첨부하도록 지시하는 역할을 합니다. 이후 단계에서 메아리가 이 지시를 처리하여 이미지를 첨부하게 됩니다.
